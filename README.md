# value-village-thrift-resell
Value Village: A Premium Thrift & Resale Platform
Value Village is a modern, full-stack e-commerce destination designed to make sustainable fashion accessible, engaging, and community-driven. It operates as a brilliant dual-sided marketplace where users can seamlessly shop for curated, pre-loved vintage items or effortlessly list their own garments for a competitive payout.

Core Features

Frictionless Dual-Marketplace: Shoppers can browse unique, handpicked items across categories like Women's, Men's, and Vintage, whilst sellers can easily set up accounts to list clothing, keeping a cracking 85% of the final sale price.

Delayed Authentication: A modern, conversion-optimised user journey that allows guests to freely explore the catalogue and view product details, only prompting them to log in or register when they attempt a secure action like adding an item to their cart or accessing the seller dashboard.

Granular Inventory Filtering: A robust, interactive filtering system allowing users to sort one-of-a-kind inventory by precise condition (e.g., Like New, Fair), size, colour swatches, and brand.

Technical Architecture

The MERN Stack: Built dynamically using MongoDB, Express.js, React, and Node.js, migrating away from a rigid static HTML architecture into a highly scalable web application.

React Router v7: Utilises advanced file-based routing and Server-Side Rendering (SSR) to deliver lightning-fast, instantaneous page transitions alongside top-tier search engine visibility (SEO).

Centralised Constraints: A rock-solid, shared data architecture bridging the frontend UI and backend API, ensuring strict e-commerce rules—such as limiting unique thrift items to a cart quantity of one—are perfectly enforced.

Design Language & UI/UX

Premium Vintage Aesthetic: The platform departs from standard, clinical e-commerce layouts by using a bespoke, warm palette of creams and rich browns, accented with bold reds and mustard yellows. Subtle texture overlays and sharp typography give the site a polished, high-street feel.

Fully Responsive: A strictly responsive approach ensures the complex product grids, authentication cards, and navigation seamlessly collapse and adapt from large desktop monitors down to mobile screens without sacrificing the user experience.
